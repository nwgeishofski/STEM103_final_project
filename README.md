# STEM103_final_project
Nicholas Wgeishofski

this is my final project for STEM103, I made a simple hangman game. To ensure that it worked properly, I tested using numbers or multiple letters, also guessing the same letter multiple times and capital letters.


While making this program I faced several challenges. The biggest 2 were understanding what things should go in which file in the program and also learning how to effectively implement objects and a class. I used 3 files for this program and previously I had only ever used 2, so understanding what things should go in which file was hard. I talked to Christine in our last class to get her tnoughts on that, also I used the internet to get some more thoughts and clarity about how it could make sense to organize my program. During this process I made 4 or 5 different algorithms and pseudocodes in my attempts to figure out how I wanted to organize everything. I used a combination of W3 schools, youtube, and AI to learn about objects and classes and how to use them effectively.
